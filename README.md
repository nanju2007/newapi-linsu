# NewAPI二开版

基于 `QuantumNous/new-api` 进行二次开发的版本。


## 二开说明

### 上游来源

本项目基于以下项目继续开发：

- `QuantumNous/new-api`

本分支是明确的二开版本，不是上游仓库的官方发行版，不保证与上游保持完全兼容。

### 主要改动方向

本次更新了
1.使用日志支持离线导出Excel/pdf
2.优化了空输入和输出问题
3.增加平均延迟
4.增加token消耗图表（输入token和缓存token）
5.支持套餐订阅
6.token输出速度
7.增加极验验证
8.忘记密码 改为了邮箱验证码重置，防止邮箱服务商拦截重置密码邮件
9.增加帮助中心
10.增加工单中心
11.增加查看收据功能
12.增加网站主题颜色设置
优化性能
流式传输错误增加详细错误提示
